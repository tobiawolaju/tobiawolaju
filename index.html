Flagship Projects ❯

SkySwarm — Autonomous Swarm Drone Platform

A fully custom-built drone system with real-time GPS tracking, onboard IMU navigation, and swarm coordination capabilities. The platform includes a custom PCB-based flight controller, real-time communication protocol (ESP-NOW/LoRa), autonomous flight software, and a React + FastAPI ground station for live telemetry, control, and geospatial tracking.

Tech Stack: C++ (ESP32), Rust (embedded), Fusion 360, KiCad, Mapbox, FastAPI, React

AgroSwarm — Autonomous Swarm of Farming Robots

An end-to-end intelligent farming system of coordinated mobile robots that clear land, plant seeds, water crops, eliminate pests, and harvest produce—entirely autonomously. This project combines AI, robotics, embedded systems, and real-time software into a scalable agri-automation platform.

Tech Stack: C++/Rust (MCUs), Python (ROS2, CV), Fusion 360, OpenCV, LoRa Mesh, FastAPI
