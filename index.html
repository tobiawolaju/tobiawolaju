Projects↗

ATOM — 5’7 Humanoid Platform ↗

Full-scale humanoid designed for low-cost fabrication and software-heavy control. Multi-sensor fusion, torque-based whole-body control, and on-board autonomy.

  • Control: Whole-body torque control + MPC gait
  • Sensing: IMU, encoders, ToF, GPS (sensor-limited design)
  • Integration: ROS 2 + custom C++ controllers + MoveIt 2
  • Powertrain: Frameless BLDC + harmonic drives
  • Focus: Performance under constrained hardware budget

Pebble Swarm — Micro Robots for Mapping ↗

Distributed micro-robotic platform for cooperative mapping and environment reconstruction. Nodes form a mesh using constrained wireless communication while maintaining collision-free motion.

  • Autonomy: decentralized pathfinding & consensus
  • Comms: multi-hop relay + fallback leader mode
  • Perception: live reconstruction using Gaussian splats
  • Goal: human-scale situational awareness from micro-agents
