Self-Driving Bot Navigation Stack

This core component focuses on robust autonomous navigation for ground robots. It integrates precise GPS data with Inertial Measurement Unit (IMU) readings using advanced sensor fusion techniques (e.g., Extended Kalman Filter) to achieve highly accurate pose estimation even in GPS-denied environments.

The system employs a multi-layered path planning algorithm (e.g., A* or Dijkstra for global path, DWA for local obstacle avoidance) and relies on finely tuned PID (Proportional-Integral-Derivative) controllers for precise velocity and heading control, enabling smooth and accurate path following. Comprehensive testing was conducted in various terrains to validate its performance and resilience to external disturbances.

Key aspects include:

  • **Sensor Data Acquisition & Filtering:** Raw data from GPS, IMU (accelerometer, gyroscope, magnetometer) is pre-processed and filtered to reduce noise and enhance accuracy.
  • **State Estimation:** Implementation of Kalman Filters for optimal estimation of robot position, velocity, and orientation.
  • **Path Planning Algorithms:** Global and local path planning strategies for efficient navigation in dynamic environments.
  • **PID Control:** Fine-tuned PID loops for motor control, ensuring the robot adheres to planned trajectories with minimal error.
  • **Obstacle Avoidance:** Integration with ultrasonic or lidar sensors for real-time dynamic obstacle detection and avoidance.

Modular Robot Chassis Design

The modular robot chassis was designed from the ground up in Fusion 360 with a focus on versatility, ease of fabrication, and rapid prototyping. Its modular nature allows for quick assembly and disassembly, making it ideal for experimental robotics platforms that require frequent component swaps or modifications.

The design prioritizes using common manufacturing methods like 3D printing (for lighter, complex parts) and CNC milling (for structural, high-strength components) to ensure accessibility and replicability. It features standardized mounting points and interchangeable sections, allowing the same base platform to be adapted for various applications (e.g., adding different sensor arrays, robotic arms, or mobility systems).

Key features include:

  • **Universal Mounting Points:** Standardized bolt patterns for easy integration of electronics, motors, and sensors.
  • **Material Optimization:** Selection of appropriate materials (e.g., PLA/PETG for 3D printing, aluminum for CNC parts) for optimal strength-to-weight ratio.
  • **Cable Management:** Integrated channels and tie-down points for neat and secure wiring.
  • **Accessibility:** Designed for easy access to internal components for maintenance and troubleshooting.
  • **Scalability:** Ability to scale the chassis dimensions up or down to accommodate different payload requirements.

Swarm Communication Protocol

Developed a custom, decentralized mesh communication protocol critical for coordinating autonomous swarm systems. This protocol leverages the strengths of both ESP-NOW (for ultra-low-latency, short-range broadcast) and LoRa (for long-range, low-power mesh networking) to ensure robust and efficient communication across multiple robot agents.

The protocol features a lightweight packet structure, dynamic routing capabilities, and built-in error checking mechanisms to maintain swarm cohesion and data integrity. It enables real-time data sharing (e.g., position, sensor readings, task assignments) and command dissemination, crucial for synchronized behaviors in complex environments.

Highlights:

  • **Hybrid Communication:** Seamlessly switches between ESP-NOW for close-proximity, high-bandwidth needs and LoRa for maintaining long-range connectivity.
  • **Decentralized Mesh:** No single point of failure, enhancing system resilience.
  • **Real-time Data Exchange:** Optimized for quick transmission of critical information between agents.
  • **Scalability:** Designed to handle an increasing number of agents without significant performance degradation.
  • **Energy Efficiency:** LoRa's low-power characteristics ensure extended operational times for battery-powered robots.

Drone Flight Controller (Custom PCB)

Designed, fabricated, and programmed a custom Printed Circuit Board (PCB) to serve as the core flight controller for autonomous drones. This custom solution provides precise control over flight dynamics, integrating various sensors and actuators directly onto a compact, optimized board.

The PCB design in KiCad focuses on minimizing size, weight, and power consumption, while ensuring robust performance. The firmware, written primarily in C++, implements advanced control loops (e.g., cascaded PID for attitude and position), sensor calibration routines, and failsafe mechanisms, demonstrating deep understanding of embedded systems and real-time operating principles.

Key Aspects:

  • **Integrated Sensors:** Onboard IMU (accelerometer, gyroscope), barometer, and magnetometer for comprehensive flight state estimation.
  • **Microcontroller Selection:** Chosen for optimal processing power and peripheral availability for real-time tasks.
  • **Power Distribution & Management:** Efficient power delivery to motors, sensors, and communication modules.
  • **Firmware Development:** Custom flight control algorithms, motor mixing, and radio communication protocols.
  • **Debugging & Calibration:** Built-in diagnostics and calibration routines for easy setup and maintenance.

Seed Dispenser & Smart Irrigation

Developed innovative mechatronic designs for precision agriculture, specifically a highly accurate seed dispensing mechanism and an intelligent irrigation system. The seed dispenser utilizes a custom-designed mechanical feeder driven by a micro-stepper motor, ensuring precise single-seed deployment at programmed intervals and depths.

The smart irrigation component integrates soil moisture sensors with solenoid-controlled water valves. An embedded algorithm analyzes moisture levels, weather forecasts, and crop-specific requirements to optimize water delivery, minimizing waste and promoting healthier plant growth. These systems are designed to be robust, energy-efficient, and easily integrated into mobile farming robots.

Components & Functionality:

  • **Precision Seed Dispensing:** Mechanical design and control logic for accurate single-seed placement.
  • **Soil Moisture Sensing:** Integration of various moisture sensor types (capacitive, resistive) for accurate data.
  • **Automated Watering:** Solenoid valve control for targeted water delivery.
  • **Intelligent Scheduling:** Algorithms for optimizing watering frequency and volume based on environmental factors.
  • **Modular Design:** Components designed for easy integration with existing or new robotic platforms.

Vision-Based Crop Analysis

Implemented a highly efficient vision-based system for real-time crop analysis, utilizing a custom-trained YOLOv8 model deployed directly onto resource-constrained microcontrollers (e.g., ESP32-CAM or Raspberry Pi). This system enables autonomous robots to perform on-the-fly identification of plant health, pest infestations, and ripeness, critical for precision agriculture.

Challenges of on-device inference were addressed through model quantization, pruning, and optimization techniques, ensuring high inference speeds with minimal computational overhead. The integration with OpenCV allows for image pre-processing, post-processing, and visual feedback, providing a powerful tool for intelligent farming and early disease detection.

Key Technologies:

  • **YOLOv8 Model Deployment:** Optimized inference of YOLOv8 for real-time object detection on embedded systems.
  • **Model Optimization:** Techniques like quantization and pruning to reduce model size and improve inference speed.
  • **OpenCV Integration:** Used for image capture, pre-processing (e.g., normalization, resizing), and post-processing (e.g., bounding box drawing, feature extraction).
  • **Hardware Acceleration:** Leveraging specific hardware capabilities of microcontrollers for faster AI operations.
  • **Actionable Insights:** Ability to trigger specific robotic actions (e.g., targeted spraying, harvesting) based on vision analysis results.
